# SECR 1213: Network Communications

## 📌 Course Overview
* **Department:** Computer Science
* **Faculty:** Computing
* **Course Code:** SECR 1213
* **Academic Session/Semester:** 20242025/1
* **Credit Hours:** 3

---

## 📖 Course Synopsis
This course will discuss the basic topics of computer network and data communications. Based on TCP/IP Internet protocol stack, the course will apply top down approach. Starts with the important and usage of computer network in commonly applications, the approach will go further detail in the technical aspect in data communication. At the end of this course, students will have an understanding and appreciation of how the network works.

---

## 💭 Reflection

### What I Learned
Taking SECR 1213 provided me with a clear roadmap of how data travels across the globe. By exploring the TCP/IP protocol suite through a top down framework, I moved from analyzing application layer behaviors down into the core transport, network, data link, and physical protocols. Understanding packet encapsulation, port addressing, routing paradigms, and error checking mechanisms helped me map out how applications seamlessly interact over public and private infrastructures.

### Challenges & Growth
The main challenge was transitioning from abstract networking models to capturing live technical data using software analysis tools. Learning how to isolate and evaluate raw traffic headers within complex network captures required high attention to detail. Tracking TCP handshakes, measuring packet loss, and analyzing subnetting boundaries pushed me to sharpen my operational logic and diagnostics capabilities when resolving communication bottlenecks.

### Future Application
The foundational principles learned throughout this course are vital to my progression in secure systems design and backend infrastructure. Knowing exactly how protocols route traffic will guide me in configuring low latency server operations and debugging distributed microservices. Moving forward, I plan to utilize these architectural baselines to design secure network topologies, minimize packet overhead, and build stable communication environments.
